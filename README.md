# Telegrambot_get_info_about_devices
This is a project to get an explanation about a device (device type, platform, functional purpose) from AI based on the received MAC/IP address and open ports.

To work with the bot, you need to get API KEYS for some resources (ABUSEIPDB, SHODAN, MACLOOKUP)

User instructions:
1. Follow the link.
2. Activate the bot / start .
3. Using the bot menu, click the "Hello" button. Then click the "Work" button.
4. Enter data
For example: "Hello, please look at this info: ip: 120.168.0.10, 00:1A:2B:3C:4D:5E another ip 10.0.0.5. Ports 22, there are 88 more, you can also look at 443"
5. Wait for the bot to analyze and read the information.
